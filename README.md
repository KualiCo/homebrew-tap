# Kuali Homebrew Tap

Homebrew formula for [Kuali Connector](https://github.com/kualico/kuali-connector) — connect Kuali to your AI assistant.

## Install

```sh
brew install kualico/tap/kuali
```

## Upgrade

```sh
brew update && brew upgrade kuali
```

## Uninstall

```sh
brew uninstall kuali && brew untap kualico/tap
```

See the [Kuali Connector documentation](https://github.com/kualico/kuali-connector) for setup and usage.
